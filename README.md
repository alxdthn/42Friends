#  42 Friends

Приложение для Android, помогающее определить местонахождение друзей в учебном корпусе школы 21.

Личный проект. Находится на стадии разработки.

Ипользованные фреймворки:
- RxJava2
- Dagger2
- Retrofit2
- Room
